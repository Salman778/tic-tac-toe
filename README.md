# Tic Tac Toe

Tic-tac-toe is a game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

### Prerequisites

Web browser

```
Google Chrome
```

## Running the Game

Open  [tic-tac-toe](https://salman778.github.io/tic-tac-toe/)

Click on any of square to start the game

Who will play first is X and in the next game will be O

Two counters for winning

Restart button for reset
